Solución a las consignas de la clase 4

  1. ¿Quién inventó el sistema de control de versión Git y por qué?

    El sistema de control de versiones GIT, fue inventado por el señor Linus Torvalds, el mismo creador de Linux, y fue desarrollado principalmente por un problema que tuvo la comunidad desarrolladora con la empresa propietaria del sistema Bitkeeper que era el utilizado por la comunidad para mantener el kernel de Linux en aquel entonces. Bitkeeper dejó de ofrecer su servicio de forma graturia por lo que Linus Torvalds emprendió su trabajo para buscar una solución a la problemática, y fue así como concibió a GIT, herramienta que no solo ha aportado al desarrollo del propio desarrollo del Kernel de Linux, si no a toda la comunidad del open source y en general a toda la comunidad de desarrolladores de software, ya que esta mágnifica herramienta permite desarrollar proyectos de forma distribuida, manteniendo además un control sobre toda la historia del proyecto para facilitar labores de solucionar potenciales problemas que se introduzcan en algún punto del proceso de desarollo.

  2. ¿A quién pertenece actualmente Github y por qué?

    Github fue adquirido por la empresa Microsoft en el año 2018, con la ascensión como CEO de Satya Nadella la compañía Microsfot ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios. Github es la piedra angular en el mundo del software libre y con este movimiento posiblemente busquen tener mayor apoyo por parte de la comunidad open source, y quizás esto sean indicios de un cambio de Mindset de Microsfot que siempre se ha caracterizado por desarrollar software propietario.

  3. ¿Hay otra forma que no sea la terminal para trabajar con Github?

    Sí, hay varias herramientas con interfaz gráfica (GUI) que permiten ejecutar las mismas acciones que se logran mediante la terminal pero todo de forma visual, a través de menús, botones y elementos visuales que facilitan la labor a personas no técnicas. La que utilicé, antes de comenzar a conocer los comandos, es una herramienta gratuita llamada SourceTree de la compañia Atlassian.
